# 📚 Book Tracker — Трекер прочитанных книг

**Автор:** Куксин Артем 

## Описание

Book Tracker — это настольное приложение с графическим интерфейсом для ведения учёта прочитанных книг. Программа позволяет добавлять книги, просматривать список, фильтровать записи и сохранять данные в файл формата JSON. Интерфейс реализован на стандартной библиотеке Tkinter, что делает приложение кроссплатформенным и не требующим установки дополнительных пакетов.

## Возможности

- Добавление новой книги (название, автор, жанр, количество страниц).
- Проверка корректности ввода (поля не пустые, число страниц — положительное целое).
- Отображение всех книг в таблице с возможностью удаления выбранной записи.
- Фильтрация списка:
  - по жанру (выбор из автоматически формируемого списка);
  - по минимальному количеству страниц (например, показать книги, в которых больше 200 страниц).
- Сохранение текущего списка в файл `books.json`.
- Автоматическая загрузка данных при запуске программы.

## Требования

- Python версии 3.6 или выше.
- Наличие модуля `tkinter` (входит в стандартную поставку Python).
